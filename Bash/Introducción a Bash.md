### Comandos de navegación

**cd**: Este comando se utiliza para moverse por los directorios de nuestro sistema. Se puede utilizar una ruta relativa (osea el directorio referenciado en el que ya nos encontramos) o una ruta absoluta (escribir la ruta completa de directorios) para movernos entre todo nuestro sistema de ficheros. También se puede utilizar `cd ..` para regresar al directorio anterior o `cd /` para iniciar a navegar desde la carpeta raíz de nuestro sistema.

**ls**: Este comando se usa para listar los ficheros dentro de un directorio, asi como ver los subdirectorios del mismo. Se puede complementar con `-a`, `-l` o `-h` para mostrar los ficheros y directorios ocultos, mostrar la información completa de cada uno y aplicar un formato human-readable al tamaño de cada fichero. Además existe un comando instalable llamado `tree` que permite visualizar lo mismo que `ls` pero de una forma un poco más ordenada y bonita visualmente.

**pwd**: Este comando sirve para ver en qué directorio nos encontramos actualmente.
